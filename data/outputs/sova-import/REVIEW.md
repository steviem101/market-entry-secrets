## Field mapping conventions

- **Slugs**: ASCII-slugified from the name, collision-checked against every
  existing slug in the target table (suffix `-2` etc. on collision).
- **Location**: Sova `State` в†’ capital-city location text (`VIC` в†’
  "Melbourne, VIC") + `location_id` from the `locations` table by state name.
  `National` в†’ "Australia" with **no `location_id`** (no national row exists);
  `TAS` likewise (no Tasmania row). Both flagged.
- **Sector tags**: Sova `Industries` в†’ `sector_vocabulary.raw_value` в†’
  `sector_slugs[]`, with verified aliases Techв†’technology, AgTechв†’agritech,
  Retailв†’retail & e-commerce, BioTechв†’healthtech (flagged). Empty industries в†’
  `sector_agnostic = true`.
- **founded/employees** (NOT NULL columns): empty string, matching the
  existing convention for unknown values.
- **investor_type**: Grant/Scholarship/Governmentв†’`grant`,
  Acceleratorв†’`accelerator`, angel-keywordв†’`angel`, Financeв†’`venture_debt`,
  else `vc`. All values already exist in the column.
- **trade_investment_agencies**: `state_body`/`state` when the host is a
  state-gov domain, else `federal_agency`/`federal`; `is_government_funded=true`.
