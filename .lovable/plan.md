## Add Crisp Chat Widget

Add the provided Crisp chat script to `index.html` immediately before the closing `</body>` tag.

### Script to embed
```html
<script type="text/javascript">window.$crisp=[];window.CRISP_WEBSITE_ID="123c2e3d-808a-425b-b1e9-b4d490d62c15";(function(){d=document;s=d.createElement("script");s.src="https://client.crisp.chat/l.js";s.async=1;d.getElementsByTagName("head")[0].appendChild(s);})();</script>
```
